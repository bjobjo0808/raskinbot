# raskinbot    

タスク - 案件毎に作成  
ログ - 勤怠データ

## 勤怠追加

### 送信

> send [対象日(yyyymmdd)] [タスク番号] [時間]

### レスポンス

勤怠を追加しました。
ログID:[ログID]
タスク名:[タスク名]
対象日:[対象日(yyyy-mm-dd)]
時間:[時間]h

### 備考

## 勤怠編集

### 送信

> edit [ログID] [対象日(yyyymmdd)] [タスク番号] [時間]

### レスポンス

勤怠を編集しました。
ログID:[ログID]
タスク名:[タスク名]
対象日:[対象日(yyyy-mm-dd)]
時間:[時間]h

### 備考

時間=0でデータ削除
